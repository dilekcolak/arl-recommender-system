# Association Rule Based Recommender System

This project implements an association rule-based recommender system using the
Online Retail II dataset.

## Objective
Generate product recommendations based on association rules derived from
German customers' transactions (2010–2011).

## Techniques
- Apriori Algorithm
- Association Rules (support, confidence, lift)
- Market Basket Analysis

## Dataset
Online Retail II (UCI / MIUUL)

## Libraries
- pandas
- mlxtend
